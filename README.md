# Student Management System

A simple Java console-based application for managing student records.

## Features

- Add Student
- Display Student Information
- Store Student Details

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## How to Run

1. Compile:

```bash
javac Details.java
```

2. Run:

```bash
java Details
```
